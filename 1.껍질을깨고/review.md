# 껍질을 깨고

1. 텍스트 파일로 자바 실행하는방법

   1. touch \_\_\_\_.java
   2. nano \_\_\_\_.java
   3. 코드작성
   4. javac \_\_\_\_.java
      - 만약 잘못된 문법이나, 오류가 발생시 컴파일되지 않고 에러를 발생시킴
   5. java \_\_\_\_
      - class 를 실행시켜줌 easy

2. 메인 메소드 안에 들어가면 컴퓨터로 하여금 무언가 하게끔 만듬
3. Math.random() 은 0~1.0 사이의 난수를 만듬 그거를 \* 원하는수 한다음 int 로 타입 캐스팅 시 랜덤넘버를 가질수 있음
4. 컴파일러 는 문법검사, 데이터 유형 등 실행과정 중 발생할수 있는 오류를 검사해줌 이걸 가상머신에서 돌리기 떄문에 빠르게 실행됨
   - 컴파일러는 보안을 해결해주지는 않는다.
